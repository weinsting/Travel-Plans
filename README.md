# Travel Plans

**A living archive of independent travel, designed and curated by TY & W.**

This is not a trip log. It is an ongoing project — a record of journeys built
around slow travel, meaningful landscapes, local culture, and memorable food,
rather than a checklist of destinations to rush through.

Every journey documented here is planned independently: self-driven routes,
deliberate pacing, and itineraries designed around a small number of places
worth staying with, instead of many places worth passing through.

> Collect moments, not things.

## About This Project

This repository doubles as the source for the public site, built intentionally
with plain HTML and CSS — no frameworks — so that it stays simple, fast, and
maintainable for years, not just for one trip.

- **Site:** [weinsting.github.io/Travel-Plans](https://weinsting.github.io/Travel-Plans/)
- **Stack:** HTML, CSS, optional vanilla JavaScript
- **Design direction:** Japanese editorial, minimal, premium typography
